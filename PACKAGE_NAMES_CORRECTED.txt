COMMON MISTAKES (AVOID THESE)
═════════════════════════════════════════════════════════════════════════

❌ WRONG - Don't do this:
  pip install mlxvlm              # Wrong - no such package
  pip install mlx-vlm-models      # Wrong - incorrect name
  pip install mlx_vlm             # Wrong - underscore not in pip name
  from mlx.vlm import load_model  # Wrong - import structure
  from mlx_vlm import load_model  # Wrong - import name is mlxvlm

✅ CORRECT - Do this:
  pip install mlx-vlm             # Correct - hyphen in pip
  from mlxvlm import ...          # Correct - import is mlxvlm
  import mlx.core as mx           # Correct - mlx.core structure
