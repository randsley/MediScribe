## Quick Math Reference

### Figma Points to Inches/MM

```

### Common Calculations

```
Full-width button with margins:
393pt (screen) - 40pt (margins) = 353pt

2-column grid with gap:
(353pt - 16pt gap) / 2 = 168.5pt each

3-column number pad:
(353pt - 24pt gaps) / 3 = 109.67pt
