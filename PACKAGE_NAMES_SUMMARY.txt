KEY FINDINGS
═════════════════════════════════════════════════════════════════════════

1. MLX Vision-Language Models (mlx-vlm)
   • Pip Install: mlx-vlm (WITH HYPHEN)
   • Python Import: mlxvlm (WITHOUT HYPHEN)
   • Status: ✅ Verified available on PyPI

2. HuggingFace Hub (huggingface-hub)
   • Pip Install: huggingface-hub (WITH HYPHEN)
   • Python Import: huggingface_hub (WITH UNDERSCORE)
   • Status: ✅ Verified (v1.3.5 installed)
   • Current Version: 1.3.5+
   • Deprecated Functions: hf_download(), cached_download()
   • Recommended: snapshot_download(), HfApi(), model_info()

3. Pattern Identified
   • Some packages use HYPHEN in pip, UNDERSCORE/NO HYPHEN in imports
   • This is common in Python packaging
   • Examples: mlx-vlm → mlxvlm, huggingface-hub → huggingface_hub

CORRECT IMPORTS
═════════════════════════════════════════════════════════════════════════

✅ CORRECT:
  import mlx.core as mx
  from mlxvlm.model_factory import load_model              # No hyphen!
  from transformers import AutoTokenizer
  from huggingface_hub import snapshot_download            # Underscore!
  from datasets import load_dataset

❌ WRONG (Don't use these):
  from mlx_vlm import load_model                           # Wrong
  from mlx.vlm import load_model                           # Wrong
  from huggingface-hub import snapshot_download            # Wrong - use underscore
  from huggingface_hub import hf_download                  # Deprecated


HUGGINGFACE_HUB FUNCTION UPDATES
═════════════════════════════════════════════════════════════════════════

Modern Functions (Use These):
  ✅ from huggingface_hub import snapshot_download
  ✅ from huggingface_hub import HfApi
  ✅ from huggingface_hub import model_info

Deprecated Functions (Don't Use):
  ❌ from huggingface_hub import hf_download         → Use snapshot_download()
  ❌ from huggingface_hub import cached_download     → Use snapshot_download()
  ❌ from huggingface_hub import hf_file_download    → Doesn't exist

Example - Modern Approach:
  from huggingface_hub import snapshot_download
  model_dir = snapshot_download(
      repo_id="mlx-community/medgemma-4b-it-4bit",
      repo_type="model",
      local_dir="./medgemma"
  )
